# WordPress Docker Compose

Despliegue de WordPress con Docker Compose y MySQL.

## Descripción
Este proyecto despliega un entorno completo de WordPress con:
- Servidor WordPress con Apache y PHP
- Base de datos MySQL
- Volúmenes persistentes para datos

## Requisitos
- Docker
- Docker Compose
